### Prerequisites

- [Visual Studio Code](https://code.visualstudio.com/)
- **Live Server** extension for VS Code

### Steps

1. **Clone the repository:**

   ```bash
   git clone https://github.com/your-username/alura-newsletter-project.git
   ```

2. **Open the project in VS Code:**

   ```bash

   ```
